# Development Guide
danilppzz development guide for colors & tech

### Tailwind
To use tailwind with HTML you can add this scripts to the head

```html
<script src="https://cdn.tailwindcss.com"></script>
<script>
  tailwind.config = {
    corePlugins: {
      preflight: false,
    },
  };
</script>
```
